# Linux Filesystem Overview - Day 3

## / 

The root of the linux file system. All files and directories start here.

## /home

Contains home directories for users. Personal files and user-level configs live here.

## /etc

Stores system-wide configuaration files. Changes here affect the entire system.

## /usr

Contains user-installed applications, libraries, and shared resources.

## /var

Holds variable data such as logs, caches, and spool files that change frequently.

## /bin vs /sbin

/bin contains essential user commands avaliable to all users.

/sbin contains system adminustration commands, usally intended for root.
