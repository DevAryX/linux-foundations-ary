APT NOTS - Day 1

apt update:
Updates the package list from Ubunti repositories.

apt upgrade:
Upgrades installed packages to newer versions.

apt install:
Installs new software packages.

apt remove:
Removes a package but keeps configuaration files.

apt purge:
Removes a package and its configuration files.

apt autoremove:
Removes unused dependencies.
